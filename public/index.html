Local Business System

About Project

This project demonstrates the interaction between frontend, backend and database technologies. It allows users to register, login, create shops, manage products and browse marketplace data.

Project Features

User Registration

Create a new account.

User Login

Secure login system.

Create Shop

Create and manage shop profile.

Add Products

Add products with images.

Browse Shops

View all shops available.

Browse Products

View all products in database.
