# **Web-Based Shop and Product Management System**

## Project Overview

Local Business System is a full-stack web application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. The project demonstrates the interaction between frontend, backend, and database technologies.

## Features

* User Registration
* User Login
* Shop Creation
* Product Management
* View All Shops
* View All Products
* Shop-wise Product Display
* Create, Read, Delete Operations using MongoDB

## Technologies Used

Frontend:

* HTML
* CSS
* JavaScript

Backend:

* Node.js
* Express.js

Database:

* MongoDB

## Project Workflow

1. User Registration – A new user creates an account by providing name, email, and password.
2. User Login – The system verifies the entered email and password from the MongoDB database and grants access after successful authentication.
3. Shop Creation – After login, the user can create a shop profile by providing shop details and image URL.
4. Shop Management – The created shop information is stored in the database and linked to the shop owner.
5. Product Addition – The shop owner can add products with product name, category, price, and image.
6. Product Management – Products are stored in MongoDB and associated with the respective shop owner.
7. View All Shops – Users can browse all available shops listed on the platform.
8. View Shop Details – Users can select a specific shop and view its details along with products belonging to that shop.
9. View All Products – Users can browse all products available in the system.
10. Product Deletion – Shop owners can remove products when required.
